## 驾驶舱 - 快速入门
### 技术约定
使用react作为view层框架，antd-design作为核心组件库, less作为css编译器

### 本地开发
在package.json中的proxy字段配置代理
```bash
$ npm i
$ npm run start
$ open http://localhost:3000/
```

### 部署
```bash
$ npm run build
```
